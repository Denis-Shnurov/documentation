API Интернет-Эквайринга

Общее описание

API Интернет-Эквайринга позволяет обрабатывать онлайн-платежи через различные каналы.

Алгоритм вычисления поля signature

Для обеспечения безопасности используется подпись запроса:

signature = HMAC-SHA256(secret_key, string_to_sign)

Калькулятор signature

Для просмотра примера расчёта:

Примеры расчёта signature на разных языках

Python


import hmac
import hashlib

secret_key = b'your_secret_key'
string_to_sign = 'example_string'

signature = hmac.new(secret_key, string_to_sign.encode('utf-8'), hashlib.sha256).hexdigest()
print(signature)

PHP


$secret_key = 'your_secret_key';
$string_to_sign = 'example_string';

$signature = hash_hmac('sha256', $string_to_sign, $secret_key);
echo $signature;

Формирование запроса на оплату

Описание

Создайте запрос для инициации платежа.

Эндпоинт

POST /payments/initiate

Параметры

Параметр Тип Описание
amount number Сумма платежа
currency string Валюта платежа

Ответ

{
    "payment_id": "12345",
    "status": "pending"
}

API Массовые Выплаты

1_Общая информация

Базовый адрес

Базовый URL для всех запросов: https://api.example.com/payouts.

Получение токена для авторизации запросов

Для получения токена используйте следующий эндпоинт:

POST /auth/token

Авторизация запросов

Добавьте полученный токен в заголовок запроса:

Authorization: Bearer YOUR_ACCESS_TOKEN

2_Работа со счётом

2.1_Получение данных о счёте по фильтру

Описание

Получите данные о счёте, применив фильтр.

Параметры
Параметр Тип Описание
account_id string ID счёта
Ответ
{
    "account": {
        "id": "12345",
        "balance": 10000.00,
        "currency": "USD"
    }
